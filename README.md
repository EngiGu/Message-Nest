# Message Nest

如果你有很多消息推送方式，每次都需要调用各种接口去发送消息，这个项目可以帮你管理各种消息方式，并提供统一的api接入。你可以自由组合各种方式，一个api推送到各种渠道，帮你省去接入的繁琐步骤。
项目名叫信息巢，意思是一个拥有各种渠道信息方式的集合站点。
